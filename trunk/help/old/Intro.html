Particle Visualizer

ParticleVis is an OpenGL visualization framework designed to load and display particle simulations. It is purely a visualization tool, completely distinct from any simulation engine that may be used to generate particle data. By loading in a flat file of particle data, you can visualize and explore the entire set in a fully realized three dimensional environment.

Notable features of PartleVis:

  • Real-time rendering of large-scale particle state data
  • Export to PNG images or AVI movies
  • Visualization of per-particle vector and scalar quantities
  • Visualization of spatial vector fields
  • Interactive color classification
  • Fully configurable geometry parser for arbitrary particle shapes
  • GLSL shader support for high-performance, high-fidelity visualization
